# Два пончика и чашка кофе — Coffee Shop Landing Page

Одностраничный лендинг для уютной кофейни «Два пончика и чашка кофе». Сайт выполнен в тёплых кофейных тонах с элегантным дизайном в стиле Apple.

**Демо:** откройте `index.html` в браузере.

---

## Технологии

- **HTML5** — семантическая разметка, `<picture>` для WebP, микроразметка Schema.org
- **CSS3** — custom properties, grid, flexbox, glassmorphism, адаптивная типографика, тёмная тема (`prefers-color-scheme`)
- **JavaScript** — бургер-меню, подсветка активного пункта навигации, Intersection Observer для анимаций, модальное окно бронирования

## Особенности

### 🎨 Дизайн
- Приятная цветовая схема на CSS-переменных
- Эффект Glassmorphism в навигации
- Адаптивный дизайн (мобильные / планшеты / десктоп)
- **Тёмная тема** — автоматически включается согласно системным настройкам
- Анимации появления при скролле (Intersection Observer)

### 🍔 Функционал
- Бургер-меню на мобильных устройствах
- Подсветка активного раздела при скролле
- **Модальное окно бронирования столика** — с формой (имя, телефон, дата, время, количество гостей, комментарий)
- **Секция отзывов** — с рейтингом и цитатами гостей
- Иконки социальных сетей (Instagram, Telegram, Facebook)

### 🖼️ Производительность
- **Lazy loading** изображений (`loading="lazy"`)
- **WebP** — современный формат с лучшим сжатием (поддержка через `<picture>`)
- **Оптимизация JPEG** — сжатие до 85% качества, прогрессивная загрузка
- **Resize** — hero до 1200px, карточки до 600px (были 4000-6000px)
- **Указаны width/height** для всех изображений — устранение CLS (Cumulative Layout Shift)

**Суммарная экономия трафика: 98.2%**  
(12.46 MB оригиналов → ~224 KB WebP / ~379 KB JPEG)

### 🔍 SEO
- Мета-теги: description, keywords, robots
- Open Graph (og:title, og:description, og:image, og:type)
- **Микроразметка Schema.org** `CafeOrCoffeeShop` (адреса, часы работы, телефон, кухня)
- Favicon через data: URI (без лишних файлов)

### ♿ Accessibility
- Skip-to-content ссылка
- `role="banner"`, `role="navigation"`, `role="contentinfo"`
- `aria-label` для всех интерактивных элементов
- `aria-expanded` для бургер-меню
- `aria-modal` для модального окна
- `focus-visible` outline для клавиатурной навигации

## Быстрый старт

Просто откройте `index.html` в браузере:

```bash
# Windows
start index.html

# macOS
open index.html
```

Никаких сборщиков и зависимостей не требуется — это чистый HTML/CSS/JS.

---

## Оптимизация изображений

Для повторной оптимизации (например, после замены изображений):

```bash
pip install Pillow
python scripts/optimize-images.py
```

Скрипт:
1. Уменьшает hero-изображение до 1200px, карточки меню до 600px
2. Сохраняет стиснутый JPEG (85%) в `images/opt_*.jpg`
3. Конвертирует в WebP (80%) в `images/webp/*.webp`
